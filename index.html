JavaScript Array Methods

Use toString() and join() to convert and array to a string.

Use pop() to remove the last element of an array.

Use push() to add a new element to the end of an array.

Use shift() to remove the first element of an array and shift all the other elements to a lower index.

Use unshift() to add a new element to the beginning of an array.

Change the third element of an array using array indexes.

Use concat() to merge two arrays.

Use concat() to merge three arrays.

Use splice() to add a new element to an array.
